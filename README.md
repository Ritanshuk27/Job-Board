** Job Board Web Application**

A fully functional Job Board Web Application developed using **HTML, CSS, JavaScript, Python (Flask), and SQLite** database.

---

## 📌 Project Overview

This project is a simple Job Board web app where:
- Employers can post jobs
- Job seekers can view job listings
- Admin can manage job postings

Built with:
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask framework)
- Database: SQLite

---

## 📂 Features

- User-friendly Interface
- Job Posting Functionality
- Job Listings Page
- Admin Dashboard (Optional)
- SQLite Database Integration
- Responsive Design (Mobile Friendly)
- Clean & Structured Code



## 💻 Installation & Setup

### Prerequisites

- Python 3.x installed
- `pip` package manager
